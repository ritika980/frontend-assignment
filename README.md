# E-Commerce Store

A clean, responsive frontend e-commerce project built with plain HTML, CSS, and JavaScript.
It includes a premium hero slider using Swiper.js, a product grid, tabs, and an accordion layout.

## Features

- Responsive hero slider with autoplay, arrows, and pagination
- Premium banner section with readable overlay text
- Product grid with cards, prices, and hover effects
- Tabs and accordion for content on different screen sizes
- Mobile-friendly layout
- High-quality banner images from Unsplash

## Files

- `index.html` - main page structure
- `style.css` - all styling and responsive layout rules
- `script.js` - Swiper initialization and UI interactions

## How to Run

Open the project in your browser using a local server.

```bash
python3 -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

## Notes

- Swiper.js is loaded from a CDN.
- Product images use local files in the project folder.
- Banner images use reliable Unsplash URLs.
