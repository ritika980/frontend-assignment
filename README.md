# Frontend Assessment — E‑Commerce Landing Page
A compact, framework-free front-end project demonstrating responsive layout and basic UI interactions: hero slider, product grid, tabs, and an accordion. Built with plain HTML, CSS, and JavaScript so you can preview it in a browser without a build step.

## Folder layout

Detailed project structure (recommended):

```
frontend-assessment/
├─ index.html                 # Main HTML entry
├─ style.css                  # Global styles
├─ script.js                  # Main JavaScript (slider, tabs, accordion)
├─ README.md                  # Project overview (this file)


Short descriptions:

- `index.html`: page markup and CDN links (Swiper).
- `style.css`: layout and responsive rules.
- `script.js`: UI behavior (slider initialization, tabs, accordion handlers).

Use relative paths when referencing images (for example `assets/images/11.svg`).

## Features

- Responsive hero slider (Swiper.js via CDN)
- Product card grid with hover states
- Tabs and accordion components for compact content
- No build step required — edit the HTML/CSS/JS directly


## Development

- Edit `index.html`, `style.css`, and `script.js` directly, then refresh the browser.
- Use VS Code Live Server for in-editor preview and auto-reload.
